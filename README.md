## Description

This a mobile app to get the current price for a specific crypto.

Project from React Native course by https://github.com/codigoconjuan

## Technologies

- React Native
- TypeScript
- Expo
- SASS
- Redux
- Redux Toolkit
- Redux Thunk

## Installation

`npm i`

## Run project

`npm start`
